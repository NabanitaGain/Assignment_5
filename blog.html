Question-2: What is the difference between innerHTML, innerText, and textContent ?

Retrieves or modifies the HTML content within an element, including tags; retrieves or modifies only the visible text, ignoring hidden elements; retrieves or modifies all text, including hidden content, without interpreting HTML.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in which a parent element manages events from its child elements using event bubbling, simplifying the handling of dynamically created elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process where an event triggered on a child element propagates upward through its parent elements in the DOM structure.
